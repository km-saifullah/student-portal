# Student Portal

## Student Portal Based on ReactJS & Firebase

### Install The Project

1. Step-1:
   Clone the repository
2. Step-2:
   Navigate to the student-portal directory
   ```
   cd student-portal
   ```
3. Step-2: Install All Packages
   ```
   npm install
   ```
4. Step-3: Run the Server
   ```
   npm run dev
   ```
